If you update a base deployment cloudformation template:
Upload the new .template file to the s3 bucket s3://cloudformation-spectrum/templates/Deployments/
Keep the title the same, but if you do change it, you'll have to change the link on the PreciselyCloudServices box, which is hosting the website

If you update a module Install template:
Make sure the changes are reflected in the resource create or modify .cli Files
Go to AWS SSM Documents and create a new version of the applicable document.
create a new Default to set it to the new version
Upload new template to s3://cloudformation-spectrum/templates/
Upload updated cli or txt documents to s3://cloudformation-spectrum/Modules/[Module]/[ModuleOS]/moduleScripts[region]
